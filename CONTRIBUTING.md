# Contributing to AI Code Review Agent

This is a Kaggle competition submission project.

## Project Structure

- `README.md` - Main project documentation
- `KAGGLE_FINAL_SUBMISSION.md` - Competition submission writeup
- `KAGGLE_WRITEUP.md` - Detailed technical writeup
- `KAGGLE_WRITEUP_SUMMARY.md` - Executive summary
- `docs/` - Additional documentation (archived)

## For Reviewers

1. Read `README.md` for quick start
2. Check `KAGGLE_FINAL_SUBMISSION.md` for full submission details
3. Run `./run_all_features.sh` to see all features
4. Launch `./start_ui.sh` for web interface
